# Tic Tac Toe

## Description

This project is a two-player Tic Tac Toe game created using Python and Tkinter. Players enter their names and choose the number of rounds to play. The game keeps track of each player's score and announces the overall champion once a player wins the required number of rounds.

## Requirements

* Python 3
* Tkinter

## How to Run

1. Save the Python file.
2. Run the program:

   ```bash
   python tic_tac_toe.py
   ```
3. Enter both players' names and the number of rounds.
4. Play the game until one player becomes the overall champion.
